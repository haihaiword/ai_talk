# ai-project

### ⚡ 环境说明

建议使用 npm, 避免出现奇奇怪怪的问题 <a href="http://nodejs.cn/" target="_blank">node 版本 > 14.18+/16+</a>

> Vite 不再支持 Node 12 / 13 / 15，因为上述版本已经进入了 EOL 阶段。必须使用 Node 14.18+ / 16+ 版本。

```bash
# 安装依赖
npm install

# 运行项目
npm run dev

# 打包发布
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
